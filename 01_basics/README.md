# 01. 기초 (Basics)

이 폴더는 컴퓨터 비전 학습을 위한 기초 개념과 환경 설정을 다룹니다.

## 📚 학습 내용

1. **Python 기초 복습** - 컴퓨터 비전에 필요한 Python 핵심 개념
2. **NumPy 기초** - 수치 연산과 배열 처리
3. **OpenCV 기초** - 이미지 읽기, 쓰기, 표시
4. **Matplotlib 기초** - 이미지 시각화

## 📁 파일 구성

- `01_python_basics.py` - Python 기초 문법 복습
- `02_numpy_fundamentals.py` - NumPy 배열 연산
- `03_opencv_introduction.py` - OpenCV 기본 사용법
- `04_image_display.py` - 이미지 표시 및 저장
- `05_basic_operations.py` - 기본 이미지 연산

## 🎯 학습 목표

- Python과 주요 라이브러리 사용법 숙지
- 이미지 데이터 구조 이해
- 기본 이미지 조작 능력 획득
- 개발 환경 구축 완료

## 🚀 시작하기

```bash
cd 01_basics
python 01_python_basics.py
```

각 파일을 순서대로 실행하면서 기초를 다져보세요!
